# 日本在住者のための有事備え完全ガイド（2026年3月）

## エグゼクティブサマリー

2026年3月時点、日本のSNS上では有事・災害への備えに関する投稿が急増している。東日本大震災15周年（2026年3月11日）を契機にした備蓄意識の再高揚、南海トラフ巨大地震への警戒、台湾有事シナリオへの現実的関心が重なり、「防災×有事」という複合的な備えの議論が活発化している。

1. **食料・水の最低7日分備蓄**（ローリングストック方式）
2. **停電対策としての蓄電池・ポータブル電源の準備**
3. **現金の手元保有**（ATM・銀行が使えなくなる前提）

5. **医薬品・衛生用品の備蓄**

軍事衝突や有事そのものへの恐怖よりも「インフラ停止・物流断絶・情報遮断」への具体的な対応策を求める声が多いのが2026年の傾向である。

## 2. 絶対やるべき備蓄チェックリスト

### 食料（品目・量・保存期間）

**基本方針：ローリングストック（普段食べるものを多めに買い、古いものから消費）**

SNS上で特に支持されているアイテム：

| 品目 | 推奨量（1人） | 保存期間 | 備考 |
|------|------------|---------|------|
| 白米（無洗米） | 10kg以上 | 精米後1〜2年 | 停電でも炊ける方法を確保すること |
| パックご飯 | 20〜30パック | 1〜2年 | 水160ccで食べられるものが便利 |
| レトルトカレー（温め不要タイプ） | 20個以上 | 1〜3年 | グリコ「カレー職人」等は常温で食べられる |
| レトルトご飯・おかず | 各10〜20袋 | 1〜5年 | 5年保存の防災食も選択肢 |
| 乾燥パスタ・うどん | 2〜3kg | 2〜3年 | カセットコンロで調理可能 |
| 缶詰（魚・肉・野菜） | 30〜50缶 | 3〜5年 | 汁ごと活用できる |
| 羊羹・チョコレート | 適量 | 1〜5年 | 防災士が推奨。水不要、カロリー高い |
| レトルトスープ・みそ汁 | 30袋 | 1〜2年 | 水分・塩分補給に重要 |
| 冷凍うどん | 適量 | 冷凍庫停電まで | 停電前に消費 |

**重要な原則**：
- 最低3日分は必須、推奨は**7日分以上**（「物流が止まると一気に物が消える」という体験談が多数）
- 「乾パンは論外」という被災経験者の声多数。美味しくて食べ慣れたものを備蓄するのが正解
- 非常食専用品は高額。普通のレトルト食品で十分

### 飲料水

| 品目 | 推奨量（1人） | 備考 |
|------|------------|------|
| ペットボトル飲料水（2L） | 7日分×3L＝21L | 1人1日3Lが目安 |
| 長期保存水（5〜10年） | 別途9L以上 | 定期点検が不要で楽 |
| 折りたたみウォータータンク | 20〜50L容量 | 地区の水源地・給水車対応 |

- 「水は100リットル備蓄してる」という投稿もあり。スペースが許す限り多めに
- 断水時は近隣の水源地や給水車に頼ることになるため**水容器（タンク）は必須**

### 医薬品・衛生用品

| 品目 | 備考 |
|------|------|
| 常備薬（市販） | 解熱剤、胃腸薬、絆創膏、消毒薬、下痢止め |
| 処方薬 | 持病がある人は「薬事法的にヤバいけど最低1ヶ月分欲しい」という投稿あり。主治医に相談して多めに処方してもらう |
| 救急セット | 包帯、三角巾、体温計 |
| 衛生用品 | トイレットペーパー、ティッシュ、マスク、アルコール消毒 |
| 非常用トイレ | 断水時に必須。凝固剤タイプを30〜50回分。猫砂でも代用可 |
| 生理用品・乳幼児用品 | 家族構成に応じて |

**薬の備え方のポイント**：
- 定期処方の薬は、次回受診前でも「残薬が少なくなった」と伝えると早めに処方してもらえる場合がある
- 市販のOTC薬は平時に揃えておく（有事になると薬局から消える）

### 燃料・エネルギー（灯油・ガス・電池）

| 品目 | 推奨量 | 備考 |
|------|------|------|
| 灯油 | ポリタンク3本以上（54L） | **ファンヒーターは停電時使えない。石油ストーブ（電気不要）が必須** |
| カセットガスボンベ | 30本以上 | カセットコンロの燃料。1本約60〜90分使用可能 |
| 乾電池（単1〜単4） | 各20〜30本 | ラジオ、懐中電灯に使用 |
| モバイルバッテリー（大容量） | 2万mAh以上 | スマートフォン充電用 |

**重要**：灯油ファンヒーターは電気を使うため停電時に使えない。東日本大震災の経験者から「石油ストーブ（電気不要型）を買っておいて良かった」という声が多数。

### 現金

- **最低5〜10万円の現金を手元に**（ATMが止まる・銀行が閉まる前提）
- ボリビアのクーデター事例（2024年6月）では「有事発生後2〜3時間でATMから2億ボリビアーノが引き出された」という事例あり。日本でも同様の事態が起きる可能性
- 小銭（100円・500円）も多めに保有しておく（停電時は電子決済・ICカード不可）
- **平時のうちに、焦らず少しずつ引き出しておくこと**

## 3. エネルギー自立（停電対策）

### 太陽光パネル・蓄電池

**SNS上の評価**：
- 「通常住宅でのソーラー＋蓄電池は、設置費用の割に短時間の停電対応が限界」という冷静な指摘も
- 「防災の日を機に家庭用ソーラー発電機を準備する必要を感じた」という声も多い
- EV（電気自動車）の「走る蓄電池」としての活用を推奨する声がある

**現実的な選択肢**：

| 機器 | 価格帯 | メリット | 注意点 |
|------|------|---------|------|
| ポータブル電源（2000Wh級） | 8〜15万円 | 冷蔵庫・医療機器も動かせる、UPS機能付きは停電時10msで自動切替 | 大容量は重い |
| ポータブル電源（500Wh級） | 2〜5万円 | スマホ充電・照明なら十分 | 大型家電は動かせない |
| ソーラーパネル（折りたたみ式） | 2〜5万円 | ポータブル電源と組み合わせで長期運用可能 | 天候依存 |
| 家庭用蓄電池+太陽光パネル | 100万円〜 | 長期停電でも自立運用可能 | 高額、設置工事必要 |

**推奨**：まず「AFERIY/Jackery等のポータブル電源2000Wh級＋ソーラーパネル」を購入するのが現実的。南海トラフ地震対策としても有効。

### 発電機

- ガソリン発電機は騒音・排気ガスで室内使用不可、ガソリン備蓄にも限界がある
- カセットガス式発電機（ホンダEU9i等）はアパートでも使いやすい
- 「南海トラフに備えるならUPS付き蓄電池が発電機より正解」という専門家的な指摘もあり

### カセットコンロ・燃料

- **全家庭に必須**。IHや電気ケトル代替として。
- カセットガスボンベは**30本以上**を推奨（湯沸かし、調理、レトルト湯煎に必須）
- 湯煎したお湯を次のレトルトにも再使用できる（水の節約）
- アウトドア用キャンプ道具（ガスバーナー、飯ごう等）は防災にも転用可能

## 4. 通信・情報収集手段

**停電・通信インフラ障害を前提とした情報収集ツール**：

| 手段 | 優先度 | 備考 |
|------|------|------|
| 乾電池式AM/FMラジオ（防水型） | 最優先 | NHKラジオはインフラ障害時でも最後まで残る媒体 |
| スマートフォン（モバイルバッテリー充電） | 高 | 通信が生きている間は最強。充電環境を確保 |
| ワンセグ対応機器 | 中 | テレビ放送の受信 |
| トランシーバー（特定小電力） | 中 | 近距離の家族間連絡、電話回線不要 |
| 衛星電話・Starlink | 低〜高 | 高額だが通信インフラ全壊時でも機能する |

**緊急連絡の準備**：
- 家族・同居人との緊急連絡手段を事前に決めておく（集合場所の確認）
- 「171」（災害用伝言ダイヤル）の使い方を家族で練習

**Jアラートへの対応**：
- Jアラートが鳴ったら：**建物内または地下に直ちに避難**（ミサイル着弾まで数分）
- 現状、Jアラートへの市民の危機意識が薄い（「どうせここまで来ない」「偵察衛星の発射でしょ」という反応が多い）
- 沖縄・南西諸島在住者は特に避難訓練を実施しておくこと
- 避難場所・避難経路の事前確認が必須

## 5. 医療・薬の備え

### 処方薬を持つ人

1. **主治医に相談**して「災害備蓄用」として多めに処方してもらう（多くの医師が対応可）
2. お薬手帳を常時携帯（避難先の医療機関でも薬が処方してもらえる）
3. ジェネリック医薬品に切り替えると入手しやすい場合がある

### 市販薬（OTC）のリスト

| カテゴリ | 具体例 |
|------|------|
| 解熱・鎮痛 | ロキソニン、カロナール、バファリン |
| 胃腸薬 | 正露丸、ビオフェルミン、ガスター10 |
| 下痢止め | ストッパ |
| 風邪薬 | パブロン、ルルなど |
| 傷・消毒 | マキロン、オキシドール、絆創膏（大量） |
| 目薬 | アレルギー対応含む |
| 抗ヒスタミン | アレルギー性鼻炎薬 |

### 衛生面での注意

- 断水時の手指衛生のために**アルコール消毒液を大量備蓄**
- 歯磨き粉・歯ブラシの予備（口腔ケアは感染症予防に重要）
- 非常用トイレは衛生上の最重要アイテム（断水時、水洗トイレは使用不可）

## 6. 資産の物理的安全

### 現金vs銀行預金

**有事・大規模災害時のシナリオ**：
- 停電→ATM停止→現金引き出し不可
- 通信障害→電子決済・クレジットカード不可
- 物流断絶→物資の現物取引に移行

**結論**：
- **現金5〜30万円を自宅に保管**（家族構成に応じて）
- 小額紙幣（千円札・五百円玉）多めに
- 銀行預金は「有事になってから引き出す」では遅い

### 「有事の金」

- ウクライナ戦争時、タイ在住者が「有事の金」を購入して資産保全した事例がSNSで話題に
- 金（ゴールド）は通貨価値崩壊時にも価値を保つ
- ただし**流動性が低い**ため、現金の代替ではなく補完手段として

### 貴重品の保管

- パスポート・権利証・保険証書等の重要書類はコピーを複数箇所に分散保管
- 非常用持ち出し袋に貴重品のコピー一式を入れておく
- パスポートは**常に有効期限内を維持**（海外脱出・帰国時の身分証明に必須）

## 7. 移動手段・避難計画

### 都市vs地方のリスク比較

| 項目 | 大都市（東京等） | 地方（農村・地方都市） |
|------|------|------|
| ミサイル・攻撃ターゲット | 高（政治・経済中枢） | 低 |
| 食料調達 | 脆弱（物流依存度高い） | 比較的強い（農地・自給能力） |
| 人口密度・パニックリスク | 高 | 低 |
| 医療インフラ | 充実（通常時） | 脆弱（有事の混雑時に機能不全も） |
| エネルギー自立 | 困難 | 比較的容易（庭・スペースあり） |

**SNS上の傾向**：「地方移住」「二拠点生活」で有事リスクを分散するという選択は理解されているが、実際に移住した人は少数。「備蓄・エネルギー自立を整えた上で現地で対応する」というスタンスが主流。

### 避難計画の基本

1. **自宅周辺の避難場所・避難経路の確認**（ハザードマップで確認）
2. **家族の集合場所を2〜3か所決めておく**（連絡が取れない場合の対応）
3. **避難袋（非常用持ち出し袋）を常に最新状態に維持**（毎年3月・9月・6月・12月に点検）
4. **車のガソリンは常に半分以上を維持**（ガソリンスタンドが停止する前に移動できる）

### 海外オプション

- **パスポートは常に有効期限内を維持する**（残存期間6ヶ月以上が望ましい）
- 海外移住先で有事が起きた場合、日本政府による邦人移送時にパスポートが必要（身分証明に時間がかかる）
- 「海外逃げ」の選択肢は現実的だが、移住先でも有事のリスクはある
- 短期的な海外避難先として、英語圏・政治的安定国（カナダ、オーストラリア、台湾以外の東南アジア等）を候補にしておく

## 11. キーポイント

1. **「備えは平時にしかできない」** — 有事になってから動いても遅い。物流が止まれば一気に物が消える。

2. **停電を最大のリスクと考える** — 現代の生活はすべて電力依存。停電が起きると水道・ATM・通信・電子レンジ・IHすべてが止まる。

3. **ローリングストックが最も合理的** — 高額な専用非常食より、普段食べているものを多めにストックして順番に消費する。

4. **石油ストーブ（電気不要型）は冬の必需品** — 電気式ファンヒーターは停電時に使えない。特に寒冷地は必須。

5. **Jアラートは「オオカミ少年」ではない** — 北朝鮮のミサイルが実際に沖縄上空を通過した事例がある。鳴ったら即、建物内または地下へ。

6. **現金は手元に置く** — 電子決済・クレジットカードは停電・通信障害で即使えなくなる。

7. **「過剰に怖がる必要はないが、具体的な備えは必要」** — 日本のインフラは脆弱な部分がある。パニックにならずに淡々と準備することが大切。

8. **パスポートは常に有効期限内を維持** — 最悪の事態では海外脱出の選択肢も残しておく。
